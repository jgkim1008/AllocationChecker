# 카카오톡 오픈채팅방 봇

카카오톡 PC 버전에서 동작하는 자동응답 봇입니다.

## 설치

```bash
cd scripts/kakao-bot
pip install -r requirements.txt
```

## 사용법

1. **카카오톡 PC** 실행
2. 모니터링할 **오픈채팅방** 열기
3. 봇 스크립트 실행:
   ```bash
   python bot.py
   ```

## 설정

`bot.py` 상단의 설정 변경:

```python
CHAT_ROOM_NAME = "오픈채팅방 이름"  # 채팅방 이름 (None이면 자동 감지)
CHECK_INTERVAL = 1  # 메시지 체크 간격 (초)
BOT_PREFIX = ["$", "&", "?", "봇", "피보"]  # 반응할 접두사
```

## 명령어

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `$종목명` | 종목 시세/분석 | `$AAPL`, `$삼성전자` |
| `&피보나치` | 지수 현황 | `&피보나치` |
| `&미국` | 미국 근접 종목 | `&미국`, `&us` |
| `&한국` | 한국 근접 종목 | `&한국`, `&코스피` |
| `&도움말` | 도움말 | `&도움말` |

## 주의사항

⚠️ **비공식 방식**입니다.
- 카카오 이용약관 위반 가능성
- 계정 정지 위험 있음
- 개인 책임하에 사용

## 24시간 실행

윈도우 작업 스케줄러에 등록하거나:

```bash
# 백그라운드 실행 (PowerShell)
Start-Process -WindowStyle Hidden python bot.py
```
